Algunos de los lenuajes de programación más utilizados en la actualidad

Java

Características:

  1. Orientado a objetos
  2. De uso general
  3. Multiplataforma
  4. Basado en C++
  5. Se puede integrar en HTML

Entorno de ejecución

Los programas escritos en Java utilizan para su ejecución una máquina virtual (JVM), pueden correr en forma de aplicaciones de escritorio, en navegadores con applets, en servidores web como serverlets o en forma de páginas activas

C#

Características:

  • Orientado a objetos
  • De uso general
  • Multiplataforma
  • Soporte a redesBasado en C++ y Java
  • Se puede icorporar a páginas HTML

Entorno de ejecución

Los programas escritos en C# comparten el marco de trabajo de Microsoft conocida como .NET Framework, el cual permite que los programas puedan interactuar en diferentes lenguajes.
